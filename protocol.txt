Ze strony klienta:
logowanie:
type: login
data: password
who: user

pobierz listę dostępnych aukcji:
type: ask
data: dload_trades
who: user

dołącz do aukcji:
type: join_trade
data: trade_id
who: user

dodaj swój przedmiot:
type: add_item
data: name, min_value, trade_time
who: user

Podbij cenę:
type: set_price
data: price, trade_id
who: user


Ze strony serwera:
Autoryzacja logowania:
type: login
data: true/false
who: user

Wyślij listę dostępnych aukcji:
type: trade_list
data: trade_list
who: user

Autoryzacja dołączenia do aukcji:
type: accept_join
data: true/false
who: user

Akceptacja nowego przedmiotu:
type: accept_item
data: true/false
who: user

Informacja zwrotna przebicia ceny:
type: set_price
data: too_low/trade_ended/accept
who: user
